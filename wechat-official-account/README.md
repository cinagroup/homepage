# 微信服务号自动化发布配置

## 📋 配置信息

请在下方填写您的微信公众号信息：

```yaml
# 微信公众号基本配置
wechat:
  app_id: "请填写您的 AppID"
  app_secret: "请填写您的 AppSecret"
  token: "请填写您的 Token"
  encoding_aes_key: "请填写您的 EncodingAESKey"
  
  # 发布配置
  publish:
    schedule: "0 */6 * * *"  # 每 6 小时执行一次
    timezone: "Asia/Shanghai"
    auto_publish: true       # 是否自动群发（true=自动发送给用户，false=仅保存为草稿）
    
  # 内容配置
  content:
    source: "ai-news-briefing"  # 内容来源
    language: "zh-CN"           # 目标语言
    translate: true             # 是否翻译为中文
    
  # 封面图配置
  cover:
    generate: true              # 是否自动生成封面图
    style: "minimalist"         # 风格：minimalist, tech, news
    size: "900x383"             # 尺寸：横图 900x383, 方图 500x500
```

---

## 🔧 获取微信公众号配置信息

### 1. 登录公众号后台
访问：https://mp.weixin.qq.com

### 2. 获取 AppID 和 AppSecret
1. 进入 **开发** → **基本配置**
2. 查看 **开发者 ID**
   - **AppID(应用 ID)**: 类似 `wx1234567890abcdef`
   - **AppSecret(应用密钥)**: 需要管理员扫码获取

### 3. 配置服务器（可选，用于接收消息）
- **URL**: `https://your-domain.com/wechat/callback`
- **Token**: 自定义令牌
- **EncodingAESKey**: 点击"随机生成"

### 4. 绑定运营者（推荐）
1. 进入 **设置与开发** → **人员设置**
2. 绑定运营者微信号
3. 运营者可协助管理公众号

## 📝 注意事项

### 群发次数限制
| 公众号类型 | 群发次数 | 说明 |
|-----------|---------|------|
| 服务号 | 每月 4 次 | 每 6 小时发布一次会超出限制！ |
| 订阅号 | 每天 1 次 | 可以每天发布 |

**⚠️ 重要：** 如果您的服务号每月只有 4 次群发机会，建议：
1. 改为 **每天发布 1 次**（如早上 8:00）
2. 或者使用 **模板消息** 推送（不占用群发次数，但有条件限制）
3. 或者将 6 小时简报 **合并为每日简报** 发布

### 内容审核
- 微信有严格的内容审核机制
- 避免敏感词、政治、暴力等内容
- AI 生成内容需人工审核后再发布

### 封面图要求
- 横图：900×383 px，大小不超过 2MB
- 方图：500×500 px，大小不超过 2MB
- 格式：JPG 或 PNG
- 确保无版权争议
