## 🚀 部署流程

### 步骤 1：填写配置
编辑 `config.yaml`，填入 AppID 和 AppSecret

### 步骤 2：测试脚本
```bash
cd /root/.openclaw/workspace/wechat-official-account
./scripts/publish-wechat-briefing.sh latest
```

### 步骤 3：配置定时任务
```bash
# 编辑 crontab
crontab -e

# 添加（每天 8 点发布）：
0 8 * * * /root/.openclaw/workspace/wechat-official-account/scripts/publish-wechat-briefing.sh latest

# 或（每 6 小时生成草稿）：
0 */6 * * * /root/.openclaw/workspace/wechat-official-account/scripts/publish-wechat-briefing.sh latest
```

### 步骤 4：验证发布
1. 登录公众号后台
2. 查看 **草稿箱**
3. 确认内容、格式、封面图
4. 手动群发测试
