## 🚀 如何开始

### 对于开发者

1. **获取测试代币**
   ```bash
   # 从水龙头获取测试代币
   curl https://faucet.cinachain.com/request?address=YOUR_ADDRESS
   ```

2. **配置开发环境**
   ```javascript
   // Hardhat 配置
   module.exports = {
     networks: {
       cinachain: {
         url: "https://rpc.cinachain.com",
         chainId: 520,
         accounts: [PRIVATE_KEY]
       }
     }
   };
   ```

3. **部署合约**
   ```bash
   npx hardhat run scripts/deploy.js --network cinachain
   ```

### 对于用户

1. **添加网络到 MetaMask**
   - Network Name: CinaChain Mainnet
   - RPC URL: https://rpc.cinachain.com
   - Chain ID: 520
   - Symbol: CINA
   - Explorer: https://scan.cinachain.com

2. **跨链资产**
   - 访问 [bridge.cinachain.com](#)
   - 连接钱包
   - 选择跨链资产
   - 确认交易

3. **参与 DeFi**
   - 访问 [swap.cinachain.com](#)
   - 添加流动性
   - 开始赚取收益
